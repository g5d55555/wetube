server 란?

인터넷에 연결된 모니터 없는 컴퓨터

소프트웨어적인 서버 = 인터넷(네트워크 )에 연결된 한 덩어리의 코드

url에 응답하고 접속을 허락하는 일

하나는 물리적으로 늘 켜져있는 컴퓨터라는 의미

인터넷에 연결된 내 접속요청에 응답하는 컴퓨터

서버란 접속을 받아주는 무언가


express - 몇줄의 코드로 서버를 만들 수 있는 nodejs에서 동작하는 유명한 프레임워크, 서버를 만드는 지름길

추천하는 이유는 새로운 버전이 많이 안나왔다 = 안정적이고 완성도가 높음


서버를 개발하는 것은

connection을 열어주고 listen해주고 파일을 처리하고 html을 전송해주고 데이터를 저장하고 form에서 데이터를 받아오고
이런것들이 대부분 반복적이기 때문에

Django(python), Laravel(php), Express(nodejs) 모두 다른 방식이지만 같은 일(서버만드는일)을 해낸다.



Express는 백엔드를위한 NodeJS 프레임 워크입니다.
React는 프론트 엔드 라이브러리입니다.
우리는 모두 NPM을 사용하여 설치합니다. 그러나 그들은 다른 일을합니다.
